# Environment Setup Instructions

## 1. Java Environment Setup

### JDK Version Used
- **Java Development Kit (JDK): 21 (LTS)**
- Vendor: Adoptium
- Reason: JDK 21 is a Long-Term Support (LTS) version and is recommended for modern Java and frameworks like Spring Boot and Quarkus.

### Verify Java Installation
Run the following command in the terminal:

```bash
  java -version
```


## Running HelloWorld.java on macOS

### Step 1: Open Terminal
- Press `Cmd + Space`
- Search for Terminal
- Open it

---

### Step 2: Create HelloWorld.java
create HelloWorld.java using code(VS Code) 
```bash
   code HelloWorld.java
```

### Step 3: Compile HelloWorld.java

```bash
    javac HelloWorld.java
```
It will create a HelloWorld.class file

### Step 4: Run HelloWorld.class
```bash
    javac HelloWorld.java
```

### Step 5: Run HelloWorld.class
after running this command you will get HelloWorld printed in the terminal
```bash
    java HelloWorld
```
